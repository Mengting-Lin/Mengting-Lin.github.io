切割的方法

  • Array.splice(start[, 切割個數[, 插入元素]]) => 返回陣列
  • let a = ["a","b","c","d"]
  • a.splice(1) 從索引 1 的位置開始，刪除所有元素（含索引1）
  • // ["c","d"]
  • Array.slice([start[, end]]) => 返回陣列, 返回元素只到end-1
  • PS. start/end為index，可為負數
  • String.split([separator[, 返回數量]])
  • separator可以是一個字符串或正則表達式
  • let str = 'The quick brown'
  • let words = str.split(' ') // words[2] => "brown"
  • let chars = str.split('') // chars[4] => "q"
  • let splits = str.split(' ',3) // ['The', 'quick', 'brown']

link vs. import

  1. link是XHTML標籤，無兼容問題；@import則是CSS的範疇，只能載入CSS，且可能會產生瀏覽器(低版本)不支援的問題。
  2. 用link呼叫CSS時，在頁面載入時同步加載；@import需先載入網頁再加載CSS。
  3. link支持利用JS控制DOM去改變樣式；@import則不支持。
